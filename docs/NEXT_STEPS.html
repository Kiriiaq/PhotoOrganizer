API publique à fournir :

# src/utils/licensing.py — squelette d'API

TRIAL_LIMIT = 10

@dataclass(frozen=True)
class LicenseState:
    has_valid_license: bool
    trial_count: int          # nombre de tris déjà consommés
    trial_remaining: int      # TRIAL_LIMIT - trial_count, plancher 0
    machine_id_short: str     # "MAC-7A3F-9C2E" pour affichage
    license_email: Optional[str]

def get_state() -> LicenseState: ...
def record_successful_organize() -> LicenseState: ...
def can_organize_now() -> Tuple[bool, LicenseState]: ...
def activate_key(key: str) -> LicenseState: ...
def get_machine_id() -> str: ...      # SHA-256(MachineGuid + VolumeSerial)

Détails d'implémentation :

  • Compteur dans %LOCALAPPDATA%\PhotoOrganizer\usage.dat au format JSON signé HMAC.
  • HMAC secret : embarqué via src/photoorganizer_pro/license/_secret.py (déjà en place et gitignored).
  • Si usage.dat est supprimé : on le recrée avec count=0 (faille assumée).
  • Si usage.dat est tampered (HMAC invalide) : on le reset à 0 et on log un warning.
  • Machine ID : SHA-256(MachineGuid + VolumeSerialNumber_C). MachineGuid via winreg, volume serial via ctypes.windll.kernel32.GetVolumeInformationW.

C Flow d'envoi des clés

C.1 — Option A : envoi manuel (10 premières ventes) P0 5 min par vente

Pas d'infra à déployer. Tenable jusqu'à ~10 ventes/semaine.

  1. Lemon Squeezy t'envoie automatiquement un email "New order #1234" à chaque vente.
  2. Tu génères la clé en local : 
    python -m src.photoorganizer_pro.license.keygen `
    --email client@example.com `
    --edition LIFE
    Sortie : PROG-LIFE-20560526-Y2xpZW50QGV4YW1wbGUuY29t-abc123...
  3. Tu réponds au client avec un template email préparé : 
    Bonjour,

Merci pour ton achat de PhotoOrganizer ! Voici ta clé de licence :

PROG-LIFE-20560526-Y2xpZW50QGV4YW1wbGUuY29t-abc123...

Comment l'utiliser :
1. Ouvre PhotoOrganizer
2. Clique sur "Activer" dans le bandeau supérieur, ou sur le bouton "Activer une licence" quand il apparaît
3. Colle la clé et clique "Activer"
4. C'est fait : utilisation illimitée sur cet ordinateur

Important : cette clé est liée au premier PC qui l'active. Pour changer de PC (réinstall Windows, nouveau disque, nouvel ordinateur), une nouvelle clé est nécessaire (politique stricte sans réémission).

Si tu rencontres un problème, réponds à cet email avec ton ID machine (visible dans la fenêtre d'activation).

Bonne organisation !
Emmanuel

C.2 — Option B : envoi automatique (au-delà de 10 ventes/semaine) P1 3-4 h setup

Cloudflare Worker (free tier suffit) qui reçoit le webhook Lemon Squeezy, génère la clé, et envoie un email via Resend.

Architecture :

Lemon Squeezy webhook (order_created)
       │
       ▼
Cloudflare Worker https://photoorganizer.workers.dev/lemonsqueezy
       │
       ├── Vérifie signature HMAC du webhook
       ├── Extrait email + order_id
       ├── Génère la clé (port JS du keygen.py, ~30 lignes)
       ├── Appelle Resend API → envoie email avec template
       └── Renvoie 200 OK à Lemon Squeezy

Pré-requis :

  • Compte Cloudflare (free tier) — cloudflare.com
  • Compte Resend (100 emails/jour free) — resend.com — ou Brevo / SendGrid
  • Le HMAC secret Pro (le même que _secret.py) à mettre en secret Cloudflare Worker (wrangler secret put HMAC_SECRET)
  • Le webhook signing secret de Lemon Squeezy à mettre aussi en secret Cloudflare Worker

À faire après les 10 premières ventes manuelles. Pas critique pour démarrer.
