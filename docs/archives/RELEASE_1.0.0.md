# PhotoOrganizer v1.0.0

## 🎉 First Stable Release

Professional Photo Organization and Management Tool.

### ✨ Features

- **Smart Organization**: Organize photos by date, location, or custom rules
- **Duplicate Detection**: Find and manage duplicate photos
- **Metadata Extraction**: Read EXIF data from photos
- **Batch Rename**: Rename photos with patterns
- **Folder Structure**: Create organized folder hierarchies

### 📦 Downloads

| File | Description |
|------|-------------|
| `PhotoOrganizer-1.0.0-release.exe` | Production build (no console) |
| `PhotoOrganizer-1.0.0-debug.exe` | Debug build (with console) |

### 🔧 Requirements

- Windows 10/11
- No installation required

### 📝 Keywords

Photo organizer, duplicate finder, EXIF metadata, batch rename, image manager
