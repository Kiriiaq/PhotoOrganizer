🧪 Validation IHM — PhotoOrganizer 2.3.0.dev0

Liste de vérification visuelle de l'interface. Pas besoin de connaissances techniques.

📖 Comment utiliser cette page (cliquer pour replier)

1. Remplir l'en-tête ci-dessous (Outil, Version, Testeur, Date, Environnement).

2. Pour chaque élément du tableau :

  • Lire la colonne « Où le trouver » pour localiser le bouton/champ dans l'application.
  • Faire l'action décrite dans la colonne « Ce qu'il faut tester ».
  • Choisir un statut : OK si tout se passe comme attendu, NOK si un comportement ne fonctionne pas, NA si l'élément n'est pas applicable (fonctionnalité absente, dépendance manquante…).
  • Noter dans « Valeur observée » ce qui s'est réellement passé.
  • Ajouter un commentaire libre si nécessaire (suggestion, anomalie, capture d'écran…).

3. Votre progression est sauvegardée automatiquement dans le navigateur. Vous pouvez fermer la page et reprendre plus tard.

4. À la fin : cliquer 📝 Générer résumé Markdown pour obtenir un texte prêt à coller dans le rapport, ou 📤 Exporter JSON pour archiver la session complète.

💡 Astuce : pour démarrer l'application, lancer PhotoOrganizer.exe (ou python src/main.py en développement). Aucune installation préalable n'est requise.
